Making software that lasts.

Purplelink LLC is a one-person software studio in Atlanta, Georgia, founded in 2026 by Benjamin Ampel. It builds native macOS and iOS apps for researchers and Apple-platform users, plus free in-browser tools for academic writing (LaTeX compilation, BibTeX validation, citation generation) that process files in memory and never store them.

Its free Scholar Utility Belt extension is used by 1,000 researchers and rated 4.5★ on the Chrome Web Store.

The LaTeX studio built for how researchers write.

ModernTex is a native macOS manuscript studio. Multi-file editing, a PDF preview that stays in sync, citation management, and submission-readiness checks — shaped around the manuscript workflow, not a generic text editor.

Journal quality badges (ABDC, FT50, UTD24, SJR), citation velocity, author metrics like h-index and g-index, academic lineage trees, and retraction alerts — added directly to Google Scholar. Rankings are bundled locally: no tracking, no accounts.
