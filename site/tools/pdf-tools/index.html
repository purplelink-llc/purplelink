    How to use these PDF tools

    1. Merge: add two or more PDFs, drag to reorder if needed, then click Merge.
    2. Split: choose a PDF and enter the pages to keep, like 1-3, 5.
    3. Compress: choose a PDF and a quality level, then download the smaller file.

    About these tools

    Merge and split run with pdf-lib directly in your browser - your files never leave your device. Compression uses Ghostscript on a hosted backend that processes the file in memory and discards it immediately.

    Do my PDFs get uploaded?
    For merge and split, no - everything happens locally. For compress, the file is sent over an encrypted connection, processed in an ephemeral container, and discarded right away.
    How much smaller will compression make my PDF?
    It depends on the file. Image-heavy PDFs shrink a lot at lower quality levels; text-only PDFs may barely change. Try "Balanced" first.
    Is there a size limit?
    Compression accepts PDFs up to 20 MB. Merge and split are limited only by your device's memory.
