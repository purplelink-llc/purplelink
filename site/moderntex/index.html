Features

Everything the manuscript needs

📄

Multi-file manuscript navigation

Navigate complex projects with root file detection and a structured sidebar. Jump between chapters, sections, and bibliography files without losing context.

⚡

Three compile modes

Fast, Live, and Full compile modes let you move at the speed you need. Live mode keeps the PDF in sync as you type. Full mode does the deep pass before submission.

📖

Synchronized PDF preview

Forward and inverse search — click the source to jump the PDF, click the PDF to jump the source. Side-by-side, always in sync.

📚

Citation management

BibTeX autocomplete that actually works. Search your bibliography by author, title, or key. Cite-as-you-type with smart suggestions from your .bib files.

🔍

Plain-language error diagnostics

LaTeX error messages are hostile by design. ModernTex translates them into clear explanations with suggested fixes, reducing the debugging spiral that breaks flow.

📦

Submission readiness checks

Automatically check anonymization, page limits, required sections, and submission packaging before you hit send. No more desk rejections over avoidable issues.

🔀

Version snapshots

Take semantic snapshots at any point — before a major revision, after peer review, before resubmission. Diff between snapshots with meaningful context.

🍎

Native macOS experience

Built in Swift for macOS 14+. Native dark and light mode, system fonts, proper keyboard shortcuts, and performance that doesn't make you wait.

How is ModernTex different from TeXShop or Overleaf?
ModernTex is a native macOS app designed specifically for the academic manuscript workflow — not a web app and not a general-purpose code editor. It combines multi-file manuscript navigation, synchronized PDF preview, BibTeX autocomplete that searches by author or title, plain-language error diagnostics, and submission-readiness checks in one coherent interface. It was designed for researchers, not programmers.
Does ModernTex support multi-file projects?
Yes. ModernTex has built-in multi-file manuscript navigation with root file detection and a structured sidebar for jumping between chapters, sections, and bibliography files without losing context.
